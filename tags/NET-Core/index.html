冬至（一）

什么是冬至？

冬至 是一个基于 .NET Core 平台的开源项目，其目标是构建一个简单易用的模块化的高性能分布式系统。

项目架构采用全新技术栈，遵循领域驱动设计(Domain-Driven Design,DDD)规范并结合命令查询职责分离(CQRS)架构实现事件驱动(Event-Driven Architecture,EDA)、事件溯源(Event Sourcing,ES)等特性。
